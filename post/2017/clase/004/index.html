Clase de sistemas de partículas - Unity3D

Clase de sistemas de partículas - Unity3D

Los parámetros mas usados son, Base, emisión y renderer.

Parámetros Base

Duración: La vida que tendrá todas las partículas.

Looping: Crea un loop por el cual crea un loop de las partículas.

Prewarm: Aparecen todas las partículas directamente (No se inician poco a poco).

Start Delay: El tiempo que tardan en empezar.

Start Lifetime: El tiempo desde que sale hasta que desaparece.

Start Speed: La velocidad inicial de las partículas.

3D Start Size: Modifica la malla para estirar las partículas.

Start Size: Tamaño inicial de las partículas.

3D Start Rotación: Modifica la rotación de las partículas.

Randomize Rotation: Rotación aleatoria.

Start Color: Color inicial.

Gravity Modifier: Gravedad para las partículas.

Simulation Space: Cambiar el pivote.

Scaling Mode: Escalar en el espacio global o loca.

Play On Awake: Reproducir cuando uses Awake.

Max particle: Numero máximo de partículas del sistema.

Auto Random Seed: Donde van a aparecer desde el punto de visión las partículas.

Parámetros Emission

Rate: Numero de partículas por segundo que se van a emitir.

Parámetros Shape

Shape: La forma que va a coger la partícula.

Angle: Angulo de las partículas

Radius: Aumenta o disminuye el radio.

Emit from: Modifica la zona de emisión.

Random Direction: Dirección aleatoria.

Parámetros Renderer

Render Mode: Billboard (Orientado a la cámara), tipo de renderizado Normal Direction: Dirección de la normal

Material: Material a renderizar

Sort Mode: Que partículas irán por delante de lo demás.

Sorting Fugde: Lo mismo

Cast Shadorws: Proyecten sombras

Receiver Shadows: Recibir sombras

Min Particle Size: Tamaño de las partículas mínimo.

Max Particle Size: Tamaño de las partículas máximo.

Sorting Layer: La capa

Order in Layer: Orden de la capa.

Billboard alignment: Alineación del billboard.

Pivot: Pivote general

Light Probes: Si se utilizan los probes

Reflection Probes: Si se utiliza reflection

Anchor override: La posición
