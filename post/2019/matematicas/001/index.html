Vector Cruzado - 2D

El producto cruzado de dos vectores 3D es un vector 3D, que está en la dirección del eje de rotación para rotar el primer vector para que coincida con la dirección del segundo vector, con el ángulo de rotación más pequeño (siempre menor de 180 grados ). Entonces, ¿qué significa “producto cruzado” de vectores 2D?

Me gusta pensar en el “producto cruzado” de dos vectores 2D como una escala, no un vector. ¿Como funciona esto? Básicamente, trate los vectores 2D como vectores 3D con sus componentes z iguales a ceros, tome su producto cruzado, y el resultado final es el componente z del producto cruzado. Así es como se ve la implementación en C ++:

// No es la implementacion mas eficiente
float cruce(const Vec2 &a, const Vec2 &b)
{
  Vec3 v1(a.x, a.y, 0.0f);
  Vec3 v2(b.x, b.y, 0.0f);
 
  return cruce(v1, v2).z;
}

Dado que los componentes z de los vectores 3D construidos a partir de los 2D son ceros, el componente x y el componente y del producto cruzado 3D son ceros. Por lo tanto, podemos optimizar aún más la implementación:

// No es la implementacion mas eficiente
float cruce(const Vec2 &a, const Vec2 &b)
{
  // Calcular el eje Z
  return a.x * b.y - a.y * b.x;
}

¿Qué tiene de especial?

Ahora hemos visto cuál es el producto cruzado de los vectores 2D matemáticamente, pero ¿qué significa geométricamente? Como se mencionó anteriormente, el producto cruzado de dos vectores 3D le da un eje de rotación para rotar el primer vector para que coincida con la dirección del segundo. Solo estamos extendiendo el espacio 2D a 3D y realizamos el producto cruzado, donde los dos vectores se encuentran en el plano XY. El vector 3D resultante es solo un eje de rotación. Sin embargo, dado que los dos vectores están en el plano XY, este eje de rotación causaría rotación solo en el plano XY, por lo que el eje siempre es paralelo al eje Z.

Si reducimos nuestra dimensión de 3D a 2D, el eje de rotación representa una rotación que es en sentido horario (CW) o en sentido antihorario (CCW). En otras palabras, el signo del producto cruzado 2D le dice si el segundo vector está en el lado izquierdo o derecho del primer vector (la dirección del primer vector está al frente ). El valor absoluto del producto cruzado 2D es el seno del ángulo entre los dos vectores, por lo que tomar el seno del arco le daría el ángulo en radianes.

El radián mide el ángulo presentado como central a una circunferencia y su medida es igual a la razón entre la longitud del arco que comprende de dicha circunferencia y la longitud del radio, es decir, mide la cantidad de veces que la longitud del radio cabe en dicho arco.

Utilicé productos cruzados de vectores 2D en un proyecto de un juego de peleas 2D (escrito en AS3), para decidir si el cursor del mouse está a la izquierda o la derecha de la pantalla y determinar en qué dirección dirigir.
