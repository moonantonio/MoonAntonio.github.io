MTodo Tool

March 9, 2017 · 1 minutos de lectura · Tags: Moon Antonio , dev

MTodo Tool es una herramienta sencilla para hacer un seguimiento del código en Unity3D MTodo analiza la sintaxis de tu código con categorías / etiquetas. Por ejemplo, supongamos que tienes una categoría de BUG, en la linea de tu código donde se encuentra ese bug, pondría : // < Categoria > < Mensaje > // BUG Aquí hay un bug. MTodo analizara todas las lineas de tu solución y recopilara todas las etiquetas que tengan una concordancia con las categorías que se estipulen dentro de la herramienta, dando un sitio central donde se concentra toda la información encontrada.
